Exemplos de Operadores de Atribuição em JavaScript

Atribuição Simples:

let x = 5;

Atribuição Condicional (Ternário):

let age = 15; let beverage = (age >= 21)? "Beer" : "Juice";

Atribuição por Adição (+=):

let y = 10; y += 5; // Equivalent to y = y + 5;

Atribuição por Subtração (-=):

let z = 20; z -= 5; // Equivalent to z = z - 5;

Atribuição por Multiplicação (*=):

let a = 4; a *= 3; // Equivalent to a = a * 3;

Atribuição por Divisão (/=):

let b = 100; b /= 10; // Equivalent to b = b / 10;

Atribuição por Módulo (%=):

let c = 10; c %= 3; // Equivalent to c = c % 3;

Atribuição por Exponenciação (**=):

let d = 2; d **= 3; // Equivalent to d = d ** 3;

Atribuição por Bitwise AND (&=):

let e = 5; e &= 3; // Equivalent to e = e & 3;

Atribuição por Bitwise OR (|=):

let f = 5; f |= 3; // Equivalent to f = f | 3;

Atribuição por Bitwise XOR (^=):

let g = 5; g ^= 3; // Equivalent to g = g ^ 3;

Atribuição por Deslocamento à Esquerda (<<=):

let h = 1; h <<= 2; // Equivalent to h = h << 2;

Atribuição por Deslocamento à Direita (>>=):

let i = 16; i >>= 2; // Equivalent to i = i >> 2;

Atribuição por Deslocamento à Direita Sem Sinal (>>>=):

let j = -16; j >>>= 2; // Equivalent to j = j >>> 2;
